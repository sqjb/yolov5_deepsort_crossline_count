# yolov5_deepsort_crossline_count
cross line count with 
* yolov5-v7.0
* deepsort
* flask and konva

## installation
```commandline
cd yolov5
pip install -r requirements.txt
pip install Flask
// aliyun cn proxy
-i http://mirrors.aliyun.com/pypi/simple --trusted-host mirrors.aliyun.com
```
## run 
```commandline
python main.py
```
